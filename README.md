# Social Network Analysis Portfolio: Peer Support and Group Formation

This repository presents a portfolio version of a social network analysis project conducted during my Master’s training in Social Psychology at Universitas Indonesia.

The original project examined the relationship between peer support, individual popularity, and group formation among psychology master’s students. The project focused on how academic and non-academic interactions could be represented as social networks and analysed using social network indicators.

## Project Overview

The study explored how peer support and interaction patterns relate to students’ positions within a social network. More specifically, it examined how discussion frequency, academic support, and non-academic interaction patterns were associated with degree centrality and clique formation.

The original research project used survey responses and interaction diary data from master’s students in psychology. To protect participant privacy, the original dataset is not shared in this repository. Any data included here are synthetic and used only to demonstrate the analytical workflow.

## Research Focus

This project focused on the following questions:

* How does discussion frequency relate to degree centrality within a peer network?
* How can peer support be represented through academic and non-academic interaction networks?
* How do students’ network positions differ across academic and non-academic contexts?
* How can clique formation help explain subgroup patterns in a student network?

## Methods

The project used social network analysis to transform interaction data into network structures.

The analytical workflow included:

* Data cleaning and preparation of edgelist data
* Construction of directed graph objects
* Academic and non-academic network comparison
* Degree centrality analysis
* Normalized degree centrality analysis
* In-strength and out-strength analysis
* Clique identification
* Network visualization
